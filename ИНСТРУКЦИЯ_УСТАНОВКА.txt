ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ИСПОЛЬЗОВАНИЮ
Генератор дневников перитонеального диализа

═══════════════════════════════════════════════════════════════

ВАЖНО! Все файлы должны находиться в ОДНОЙ папке:

1. index.html - главный файл приложения
2. styles.css - файл со стилями
3. script.js - файл с логикой приложения
4. html2pdf.bundle.min.js - библиотека для экспорта в PDF

═══════════════════════════════════════════════════════════════

КАК ЗАПУСТИТЬ:

1. Убедитесь, что все 4 файла находятся в одной папке
2. Откройте файл index.html в браузере (двойной клик)
3. Приложение должно открыться и работать полностью БЕЗ интернета

═══════════════════════════════════════════════════════════════

ПРОВЕРКА РАБОТОСПОСОБНОСТИ:

Если приложение не работает или нет стилей:

1. Откройте консоль браузера (F12 или Ctrl+Shift+I)
2. Посмотрите, есть ли ошибки в консоли
3. Убедитесь, что все файлы находятся в одной папке
4. Проверьте, что имена файлов написаны правильно:
   - index.html (с маленькой буквы)
   - styles.css (с маленькой буквы)
   - script.js (с маленькой буквы)
   - html2pdf.bundle.min.js (с маленькой буквы)

═══════════════════════════════════════════════════════════════

ЕСЛИ НЕ РАБОТАЕТ:

1. Убедитесь, что файл открывается через браузер (не блокнот)
2. Попробуйте открыть в другом браузере (Chrome, Firefox, Edge)
3. Проверьте, что все файлы скопированы полностью
4. Попробуйте открыть index.html через проводник Windows:
   - Правой кнопкой на index.html
   - Выбрать "Открыть с помощью" → ваш браузер
